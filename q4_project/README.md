## 方法（第四问扰动的是"偏好"，不是参数）
1. 在权重单纯形（w_R+w_dP+w_σ=1，均≥0）上密集采样约 1300 组权重。
2. 每组权重下，对 min-max 归一化后的三指标做"加权和"，在问题三的 Pareto 前沿上选最优方案。
3. 看最优 (β,γ,N_r) 随权重漂移。
4. **鲁棒判据 = win-share**：某方案"被选为最优的权重组合占比"。占比最大者 = 在最宽偏好范围内都最优 = 对偏好变化最不敏感的鲁棒设计。

为什么 min-max 归一化：三指标范围悬殊，不归一化加权和会被范围大的 dP* 主导。

## 本次结果
**鲁棒设计：β=0.22，γ=4.5，N_r=4**，win-share ≈ 19.5%（单一方案占比最高），
且 win-share 前 5 名全部聚集在 γ=4.5、N_r=4、β≈0.21–0.22 —— 鲁棒区域高度集中。

典型场景最优：均衡/重压降/重均匀时均稳定在 γ=4.5、N_r=4、β≈0.21–0.22；
仅"重热阻 R"时 γ 下移到 ≈3.25、β 升到 ≈0.23（与机理一致：低 γ → 微通道更高 → 热阻更低但压降升高）。

该鲁棒解与问题三的膝点/理想点一致 —— 第三问(客观折中)与第四问(偏好稳健)相互印证。
