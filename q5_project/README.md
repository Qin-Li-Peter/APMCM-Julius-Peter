## 关键物理判断（为什么连续扰动只作用于 β、γ）
加工误差作用在**连续几何量**上：针肋直径→β、各层高度→γ。而 **N_r 是整数排数**，制造时定死（不会做出"3.7 排"）。因此：
- 蒙特卡洛 & Sobol 只对 β、γ 施加连续加工公差，**N_r 固定**；
- N_r 另做"少一排/多一排"的**离散稳健性检查**单独报告。

这样局部灵敏度、蒙特卡洛、Sobol 三者口径一致（都针对连续公差），结论不再自相矛盾。
（早期把 N_r 当连续变量做 Sobol 会得到"N_r 主导"的错误结论，与局部灵敏度矛盾——已修正。）

## 三件分析
1. **局部灵敏度** S=(∂Y/∂x)(x/Y)：归一化斜率，|S| 大=敏感。
2. **蒙特卡洛**：β,γ 各 ±5%(1σ) 高斯扰动，N=20000，给三指标分布（均值/标准差/CV/95分位/最坏）。
3. **Sobol(β,γ)**：方差分解，定量给出 β、γ 对各指标波动的贡献占比。
4. **N_r 离散检查**：N_r±2 对三指标的相对变化。

## 本次结果（评估鲁棒解 β=0.22, γ=4.5, N_r=4）
基准性能：R\*=0.741，ΔP\*=0.085，σ_T\*=0.772。

| 分析 | 结论 |
|---|---|
| 局部灵敏度 | γ 对所有指标影响最大（对 ΔP\* 的 S≈−0.90）；β 次之，主要影响 ΔP\* |
| 蒙特卡洛 CV | R\* 0.20%、σ_T\* 0.21%（**极稳**）；ΔP\* 3.23%（最敏感，最坏 +30%） |
| Sobol 总效应 | γ 主导各指标方差（R\* 0.95、ΔP\* 0.75、σ\* 0.79），β 次之 |
| N_r±2 | 主要影响 ΔP\*（−10%/+14%），对 R\*、σ\* 影响小 |

**稳定性结论**：该鲁棒解的热阻与温度均匀性对加工误差**极不敏感**（CV≈0.2%），可制造性好；
**压降是最需要关注的指标**，且主要受 **γ（歧管/微通道层高比）** 控制 —— 与问题一机理一致
（ΔP 中歧管损失 ∝ 1/γ²）。因此**层高公差是加工控制的重点**。
